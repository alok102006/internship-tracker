# Internship Tracker

A lightweight productivity web application designed to help students stay consistent while applying for internships.
The tracker records daily applications, maintains a streak system, and provides a simple checklist so users can build discipline during their internship search.

## How It Works

The application stores user activity locally in the browser using LocalStorage.
Each day the checklist resets automatically while preserving the user's streak count. The progress bar updates in real time as tasks are completed, helping users visualize daily effort.

No backend or account is required ‚Äî the app is fast, simple, and privacy-friendly.
